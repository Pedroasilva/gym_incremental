# 🏋️ From Scrawny to Swole

An incremental game about a scrawny guy who starts weak at the gym and trains his
way to winning the **Arnold Classic**.

Pick an exercise, choose a weight, and fill the **Effort Bar** — each rep starts
hard and gets incrementally easier. Build strength, manage **hunger** and
**health**, shop the **market** (gear, vitamins, anabolics), and compete in
bodybuilding shows judged on mass, symmetry and conditioning.

**Play:** https://Pedroasilva.github.io/gym_incremental/

## Tech

- Vite + TypeScript, no runtime dependencies
- State saved to `localStorage`
- Auto-deployed to GitHub Pages via GitHub Actions

## Develop

```bash
npm install
npm run dev      # local dev server
npm run build    # production build to dist/
```
